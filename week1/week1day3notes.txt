String:
======
 It is a non primitive DataType
 It is a class
 String is a sequence of characters

 Two ways to declare String:
===========================
   String literal
     -String Constant pool
     -to use the memory space effectively
   String Object
     -Heap memory
     -will always create a new memory space

 Methods of String:
 =================
    length()-is a method to find the number of characters in a String    
    charAt()-to retrieve a particular character in the given position
    tocharArray()-To convert String to a character array
    equals()-To compare two string values
    equalsIgnoreCase()-To ignore Case sensitivity
    indexOf()-to find the index position of the given character
    contains()-to check for the partial match
    substring()-to return a string that is a substring of this string
